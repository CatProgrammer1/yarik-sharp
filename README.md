# Yarik#(YKS, Yarik Sharp)
## универсальный, лучший из лучших, интерпретруемый язык програмированния во всём мире.

### Острый Ярик может похвастатся:
* своим лёгким синтаксисом
* возможностю к созданию структур и их экземпляров
* возможность работать с поинтерами различных типов данных.
> [!Note]
> За недавними новостями, ядро линукса было переписано на Остром Ярике из-за своей эффективности.
